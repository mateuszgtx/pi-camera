# CEGŁA Camera STL Files

This directory contains optional 3D-printable enclosure files for the CEGŁA Pi Camera project.

## Files

| File | Description |
| ---- | ----------- |
| `button_main_part1.stl` | Main button part |
| `button_main_part2.stl` | Main button part |
| `button_power.stl` | Power button |
| `cam_back.stl` | Back enclosure part |
| `cam_display_cover.stl` | Display cover |
| `cam_front.stl` | Front enclosure part |
| `cam_mid.stl` | Middle enclosure frame |
| `cam_mount.stl` | Camera mount |

## Suggested print settings

These values are starting points. Adjust them for your printer, filament and part orientation.

| Setting | Suggested value |
| ------- | --------------- |
| Material | PLA or PETG |
| Layer height | 0.2 mm |
| Infill | 20-30% |
| Nozzle | 0.4 mm |
| Supports | As needed, depending on part orientation |

## Assembly notes

These models are intended for a Raspberry Pi camera build with a small display, camera module and physical buttons.

Before printing all parts, print one part first to verify fitment with your display, Raspberry Pi board, camera module, button caps and mounting hardware.
